# V0.12.1 — consolidation du compagnon UI

## Objectif

La V0.12.1 vise à consolider techniquement le plugin compagnon `IliasTraxEventBridgeCourseUI` après la stabilisation fonctionnelle de la V0.12.0.

Cette version ne crée pas de nouvelle fonctionnalité pédagogique. Elle rend la base V0.12 plus propre, plus maintenable et plus simple à installer.

La V0.13 reste réservée à la roadmap suivante : analyse IA optionnelle des traces xAPI.

## Branche de travail

```text
v0.12.1-consolidation-ui-companion
```

## Version de développement

```text
0.12.1-dev
```

La version finale sera `0.12.1` après validation et promotion.

## Problème traité

La V0.12.0 fonctionne, mais plusieurs évolutions du compagnon UI étaient appliquées par patchers successifs lors de l'installation :

- navigation Delos ;
- route Info ;
- lecture directe TRAX/LRS ;
- vues LRS primaires ;
- configuration outbox technique ;
- dashboard pédagogique ;
- apprenants en difficulté ;
- export CSV Expert enrichi ;
- layout V0.12 ;
- route PDF du tableau de bord.

Ce fonctionnement était acceptable pour stabiliser rapidement la V0.12, mais il compliquait la maintenance à moyen terme.

## Objectif technique V0.12.1

Réintégrer proprement les modifications V0.12 stabilisées dans les templates du plugin compagnon :

```text
companion/IliasTraxEventBridgeCourseUI/plugin.php.tpl
companion/IliasTraxEventBridgeCourseUI/classes/*.php.tpl
```

Le script d'installation doit continuer à installer le compagnon, mais sans rejouer les patchers historiques de l'écran cours.

## Principe d'architecture conservé

```text
Outbox locale = file technique d'envoi
TRAX/LRS      = source officielle du suivi xAPI pédagogique
```

La V0.12.1 ne modifie pas ce principe.

## Périmètre V0.12.1

### Inclus

- intégration directe dans les templates des modifications V0.12 stabilisées ;
- consolidation de la route PDF `exportCourseDashboardPdf` dans le template UIHookGUI ;
- réduction du nombre de patchers nécessaires après installation ;
- simplification du script `install_course_ui_companion_with_standalone_fix.sh` ;
- conservation stricte du rendu fonctionnel V0.12 ;
- documentation de la nouvelle procédure d'installation ;
- validation de non-régression sur VM ILIAS.

### Hors périmètre

- analyse IA des traces xAPI ;
- nouveaux indicateurs pédagogiques ;
- modification du modèle de données ;
- modification du principe TRAX/LRS comme source pédagogique ;
- suppression des diagnostics V0.11 ;
- refonte graphique majeure.

## Consolidation réalisée

Le template principal de l'écran cours a été consolidé :

```text
companion/IliasTraxEventBridgeCourseUI/classes/class.ilIliasTraxEventBridgeCourseUIScreen.php.tpl
```

Il contient désormais directement les blocs stables issus de la V0.12 :

- tableau de bord pédagogique ;
- synthèse pédagogique ;
- lecture directe TRAX/LRS ;
- analyse enrichie ;
- statuts pédagogiques ;
- apprenants en difficulté ;
- export CSV Expert enrichi ;
- layout V0.12 ;
- supervision technique outbox.

Le template UIHookGUI contient également directement la route :

```text
exportCourseDashboardPdf
```

## Simplification de l'installation

Le script principal d'installation a été simplifié :

```text
scripts/install_course_ui_companion.sh
```

En V0.12.1, il installe le compagnon depuis les templates consolidés et ne rejoue plus les anciens patchers appliqués sur l'écran principal ou sur la route PDF.

Le wrapper historique reste disponible pour compatibilité avec les procédures existantes :

```text
scripts/install_course_ui_companion_with_standalone_fix.sh
```

Il appelle désormais simplement le script principal d'installation.

## Script d'audit V0.12.1

Un script d'audit permet d'objectiver l'état du compagnon installé :

```text
scripts/audit_course_ui_companion_v0121.sh
```

Il permet de vérifier :

- la présence du template source ;
- la présence du fichier compagnon actif installé ;
- la version et la branche du plugin ;
- les marqueurs fonctionnels V0.12 présents dans le fichier actif ;
- les marqueurs présents dans le template source ;
- les patchers encore référencés par le wrapper d'installation ;
- la syntaxe PHP du compagnon ;
- un extrait de diff entre template et fichier actif.

### Exécution sur VM ILIAS

```bash
cd /var/www/ilias/public/Customizing/global/plugins/Services/EventHandling/EventHook/IliasTraxEventBridge

git fetch origin v0.12.1-consolidation-ui-companion
git reset --hard FETCH_HEAD
rm -f FETCH_HEAD

bash scripts/install_course_ui_companion_with_standalone_fix.sh
bash scripts/audit_course_ui_companion_v0121.sh
```

### Sortie attendue

Le script doit confirmer :

```text
Active V0.12 markers   : OK partout
Template V0.12 markers : OK partout
Patchers referenced by wrapper : bash "$SCRIPT_DIR/install_course_ui_companion.sh"
PHP syntax check : OK
Diff summary template vs active screen : vide
```

## Critères d'acceptation

La V0.12.1 est validée si :

- le plugin indique `0.12.1-dev` en version de validation, puis `0.12.1` en version finale ;
- le compagnon s'installe depuis les templates consolidés ;
- les vues `Tableau de bord`, `Analyse`, `Expert` et `Configuration` restent fonctionnellement identiques à la V0.12.0 ;
- la synthèse pédagogique reste visible dans `Tableau de bord` et `Analyse` ;
- les statuts `OK`, `À surveiller`, `Critique`, `Désactivée` restent cohérents ;
- les couleurs orange et rouge restent visibles ;
- l'export PDF reste fonctionnel ;
- l'export CSV Expert conserve les colonnes pédagogiques V0.12 ;
- le bloc `Apprenants en difficulté` reste présent ;
- les diagnostics V0.11 restent accessibles ;
- le script d'installation reste idempotent ;
- les contrôles `php -l` sont OK.

## Décision de versionnement

La consolidation UI est une correction technique de la V0.12.0. Elle est donc versionnée en V0.12.1.

La V0.13 reste disponible pour la fonctionnalité prévue dans la roadmap : analyse IA optionnelle des traces xAPI.
