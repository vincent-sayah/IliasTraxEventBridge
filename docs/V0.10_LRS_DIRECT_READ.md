# V0.10 — Lecture directe TRAX / LRS

## Objectif

Ajouter une lecture directe du LRS TRAX depuis l'écran `Suivi xAPI` du cours, sans remplacer l'analyse locale V0.9.1.

La V0.9.1 exploite l'outbox locale `evnt_evhk_itxeb_out`. La V0.10 ajoute un premier contrôle direct côté LRS afin de vérifier que les statements envoyés existent bien dans TRAX.

## Principe retenu

La lecture directe est volontairement séparée de l'envoi :

- l'envoi continue d'utiliser l'outbox locale et le client d'envoi existant ;
- la lecture utilise un nouveau client dédié, read-only ;
- aucune ligne SQL locale n'est modifiée pendant cette lecture ;
- l'écran ne doit pas être bloqué si TRAX est indisponible.

## Client ajouté

```text
classes/class.ilIliasTraxEventBridgeLrsReadClient.php
```

Ce client effectue uniquement des appels :

```text
GET /statements
```

avec les en-têtes xAPI existants :

```text
Accept: application/json
X-Experience-API-Version: 1.0.3
```

et l'authentification Basic HTTP déjà configurée pour TRAX.

## Synthèse cours ajoutée

```text
classes/class.ilIliasTraxEventBridgeLrsCourseSummary.php
```

La synthèse interroge le LRS avec l'activité xAPI du cours :

```text
<ILIAS_BASE_URL>/xapi/activity/course/ref/<course_ref_id>
```

Paramètres utilisés :

```text
activity=<activité cours>
related_activities=true
since=<date début période>
limit=100
```

Le paramètre `related_activities=true` est nécessaire car les statements générés par le plugin utilisent le cours comme activité parente dans `contextActivities.parent`.

## Affichage UI

Un bloc minimal est ajouté dans le tableau de bord :

```text
TRAX / LRS direct
```

Il affiche :

- état disponible / indisponible ;
- code HTTP ;
- nombre de statements retournés ;
- activité cours xAPI utilisée ;
- erreur éventuelle.

## Installation

La génération du companion applique désormais :

```text
scripts/patch_course_ui_lrs_direct_summary.php
```

via :

```bash
bash scripts/install_course_ui_companion.sh
```

ou via le wrapper Delos :

```bash
bash scripts/install_course_ui_companion_with_standalone_fix.sh
```

## Limites du lot 1

Ce lot ne remplace pas encore l'analyse locale. Il ajoute un contrôle LRS minimal.

Les enrichissements prévus pour les lots suivants sont :

- comparaison local outbox / LRS ;
- détail par ressource côté LRS ;
- indicateurs de divergence ;
- export expert enrichi local + LRS.
