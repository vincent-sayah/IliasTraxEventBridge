# V0.22.1 — Présentation type formulaire ILIAS

## Objectif

Améliorer la lisibilité du tableau de bord et de l'analyse en rapprochant la présentation du standard visuel ILIAS :

```text
libellé / fonctionnalité à gauche
contenu / données à droite
```

## Problème constaté

Les blocs du tableau de bord sont bien délimités, mais les titres et les données restent souvent empilés verticalement.

Résultat :

- les titres se distinguent mal des données ;
- la lecture est moins proche des formulaires ILIAS ;
- les blocs longs donnent une impression de liste continue.

## Solution proposée

Ajouter une mise en page réutilisable :

```text
itxeb-ilias-row
  gauche : itxeb-ilias-label
  droite : itxeb-ilias-value
```

Exemple visuel attendu :

```text
Indicateurs clés        [cards KPI]
Résumé                  [synthèse pédagogique]
Données                 [histogramme / tableau]
Vue quotidienne          [activité dans le temps]
```

## Périmètre V0.22.1

Le script applique cette structure sur les blocs principaux du tableau de bord :

- indicateurs clés ;
- synthèse pédagogique ;
- blocs avec histogrammes ;
- activité dans le temps si la V0.22 a déjà été appliquée.

La structure est prévue pour être étendue ensuite aux tableaux détaillés de l'onglet Analyse si besoin.

## Script d'application

```text
scripts/apply_v0221_ilias_like_dashboard_layout.php
```

Ce script :

1. ajoute les classes CSS `itxeb-ilias-row`, `itxeb-ilias-label`, `itxeb-ilias-value` ;
2. ajoute la méthode `renderIliasLikeRow()` ;
3. restructure les KPI du tableau de bord ;
4. restructure la synthèse pédagogique ;
5. restructure les blocs de type barres ;
6. adapte l'activité dans le temps si `apply_v022_activity_timeline.php` a déjà été lancé ;
7. passe les versions en développement à :

```text
plugin principal : 0.22.1-dev
companion UI     : 0.8.7
```

## Ordre conseillé d'application

```bash
php -l scripts/apply_v022_activity_timeline.php
php scripts/apply_v022_activity_timeline.php

php -l scripts/apply_v0221_ilias_like_dashboard_layout.php
php scripts/apply_v0221_ilias_like_dashboard_layout.php
```

## Validation attendue

Dans ILIAS :

```text
Pilotage xAPI > Tableau de bord
Pilotage xAPI > Analyse
```

Vérifier :

- les intitulés principaux sont alignés à gauche ;
- les données sont alignées à droite ;
- les KPI restent lisibles ;
- le bloc `Activité dans le temps` reste compact ;
- les affichages restent corrects en largeur réduite.

## Limite volontaire

Cette version ne modifie pas toute la logique métier.

Elle agit d'abord sur le rendu visuel afin de tester le confort de lecture avec le style ILIAS avant de généraliser aux autres tableaux complexes.
