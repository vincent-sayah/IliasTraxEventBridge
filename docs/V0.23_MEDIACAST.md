# V0.23 — Suivi MediaCast

## Contexte

ILIAS MediaCast peut contenir :

```text
- des vidéos internes servies par ILIAS ;
- des médias externes, par exemple YouTube ou Vimeo.
```

Avant la V0.23, le plugin captait l'ouverture générale de l'objet MediaCast, mais pas le détail des médias réellement consultés.

## Besoin fonctionnel

```text
1. Tracer l'ouverture de l'objet MediaCast.
2. Tracer le lancement d'une vidéo interne.
3. Tracer l'ouverture d'un média externe.
4. Afficher les vidéos et médias vus dans l'onglet Analyse.
```

## Périmètre volontairement exclu

La V0.23.8 ne trace pas :

```text
- pause ;
- reprise ;
- temps exact de visionnage ;
- progression vidéo ;
- complétion vidéo ;
- pourcentage de média vu.
```

Ces éléments peuvent faire l'objet d'une version ultérieure.

## Détection côté navigateur

Le plugin compagnon `IliasTraxEventBridgeCourseUI` injecte un JavaScript sur les pages MediaCast.

Le script observe :

```text
- les événements play sur balise video ;
- les sélections dans la playlist MediaCast ;
- l'apparition d'iframe/embed externes ;
- les titres issus de la playlist MediaCast.
```

## Beacons MediaCast

Le navigateur envoie un beacon HTTP vers l'URL courante avec des paramètres dédiés :

```text
itxeb_mcst_event
itxeb_mcst_media_id
itxeb_mcst_media_title
itxeb_mcst_media_mime
itxeb_mcst_media_url
itxeb_mcst_media_provider
itxeb_mcst_v
```

Valeurs principales de `itxeb_mcst_event` :

```text
media_played
external_media_opened
```

## Génération xAPI

Le plugin principal convertit ces beacons en statements xAPI.

### Vidéo interne

```text
verb_id = .../xapi/verbs/played-media
media_client_event = media_played
media_provider = ilias
```

### Média externe

```text
verb_id = .../xapi/verbs/opened-external-media
media_client_event = external_media_opened
media_provider = youtube | vimeo | external
```

## Déduplication

La V0.23 ajoute une déduplication pour éviter plusieurs statements identiques lors :

```text
- d'un même clic playlist ;
- de l'apparition répétée d'une iframe ;
- d'un même événement navigateur répété ;
- d'un beacon rejoué dans la même requête.
```

## Agrégation TRAX/LRS

`class.ilIliasTraxEventBridgeLrsCourseSummary.php` agrège les statements MediaCast lus depuis TRAX/LRS dans :

```text
by_mediacast_media
summary.mediacast_internal_played
summary.mediacast_external_opened
summary.mediacast_media_total
summary.mediacast_media_unique
summary.mediacast_media_learners
```

## Affichage formateur

Le bloc `Médias MediaCast vus` est affiché uniquement dans `Analyse`.

Colonnes :

```text
Média
Type
Actions
Apprenants
MediaCast
Dernière trace
```

Exemples :

```text
vid5.mp4                  | Vidéo interne | 2 lancement(s)
Hold On to the Light      | Média externe | 4 ouverture(s) externe(s)
```

## Vue Expert

La vue Expert conserve la vision technique complète des statements, avec les verbes :

```text
played-media
opened-external-media
```
